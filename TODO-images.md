# Previous Event Images - Where to Put

## Folder
Use:
- `public/images/previous-events/`

## Filenames (recommended)
- `prev-1.jpg`
- `prev-2.jpg`
- `prev-3.jpg`

## Where to reference them
In `public/index.html`, inside the section:
- `section id="previous-events"`

Replace these image URLs in the 3 gallery cards with:
- `src="images/previous-events/prev-1.jpg"`
- `src="images/previous-events/prev-2.jpg"`
- `src="images/previous-events/prev-3.jpg"`
